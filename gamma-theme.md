# Accelerate Data - Gamma Theme Configuration

Based on the Accelerate Data Brand Book, here are the values to configure your Gamma theme.

---

## Colors

### Theme Palette

| Setting | Value | Notes |
|---------|-------|-------|
| **Primary accent color** | `#00b4d8` | Pacific - embodies technology and innovation |
| **Secondary accent color 1** | `#00dd92` | Seafoam - uplifting energy and creativity |
| **Secondary accent color 2** | `#03045e` | Navy - depth and stability |

### Text

| Setting | Value | Notes |
|---------|-------|-------|
| **Heading color** | `#03045e` | Navy - for strong visual hierarchy |
| **Body color** | `#171c21` | Smoke - dark neutral for readability |

---

## Background

| Setting | Value | Notes |
|---------|-------|-------|
| **Card background color** | `#FFFFFF` | White for clean, professional appearance |
| **Page background** | None (or `#f2f2f2` Pearl for subtle warmth) | Neutral backdrop per brand guidelines |

### Body

## Design

### Cards

| Setting | Value | Rationale |
|---------|-------|-----------|
| **Roundness** | Option 2 (subtle rounding) | Clean geometric lines per brand personality |
| **Card shadow** | Option 2 (medium) | Premium, understated depth |
| **Card border** | Option 2 (thin line) | Subtle technical detail |
| **Card border color** | `#90e0ef` (Arctic) | Secondary color for subtle definition |
| **Card transparency** | Option 1 (solid/opaque) | Professional, clear presentation |
| **Accent image shape** | Option 1 (standard rectangular) | Minimalist, functional approach |

### Blocks & Content

Use default settings with brand colors applied.

### Buttons & Links

| Element | Color |
|---------|-------|

| **Button text** | `#FFFFFF` (White) |
| **Links** | `#0077b6` (Ocean) |
| **Link hover** | `#00b4d8` (Pacific) |

## Complete Color Reference

---

## Brand Voice Reminder

When creating presentations with this theme, maintain the brand personality:
- Calm, authoritative, systems-minded
- Precise and rational communication
- Minimalist, premium, purely functional design
- Strip away ambiguity; simplify complex structures
