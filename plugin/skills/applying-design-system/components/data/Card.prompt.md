Generic surface container for grouping content — the base panel for everything from feature blocks to settings sections.

```jsx
<Card>
  <h3>Governance built in</h3>
  <p>Column-level access controls and full lineage.</p>
</Card>
<Card interactive onClick={open}>…</Card>
```

White (or Smoke-raised) surface, 1px border, 8px radius, minimal shadow. Set `interactive` for hover elevation on clickable cards.
